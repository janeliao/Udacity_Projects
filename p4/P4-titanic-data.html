一、提出问题¶

泰坦尼克号¶

有哪些因素会让船上的人死亡率更高？¶

三个因素来分析:¶

1.Pclass：船票类型（社会经济地位的代表）是否影响；

2.Age：年龄是否影响；

3.SibSp：结合船票类型和家庭关系是否影响；

# 获取3种船票类型的总数。
def passenger_count(data,name):
    return data.groupby(name)['PassengerId'].count()


# 计算死亡率：船票类型为1的死亡人数 除以 船票类型为1的总人数 = 船票类型为1的死亡率。
def deceased_rate(name):
    total_passenger_count = passenger_count(titanic_datas_df, name)
    num_deceased = passenger_count(deceased_passenger_df, name)
    return num_deceased / total_passenger_count

四、结论：¶

对死亡率影响最大因素是船票类型（即社会地位），家庭关系在船票类型1的表现较为突出，年龄较前面两者之间为较少影响因素。¶

局限性：¶

数据集中存在的缺失值，我使用平均值填充缺失值，这样会影响最终结果最小。这里并没有从统计上分析得出这些结果的偶然性，所以并不知道这里的结果是真正的差异造成的还是噪音造成的。该样本的数据还是比较少，而且数据的准确性有待怀疑，导致结论还不能得以证明而且单从数据进行分析我们只可以得到相关性，而不能得到因果性。¶
